MERCHANT-THEORY-FORMULA

A Python-based implementation of ideal Otto Cycle thermodynamic formulas using user-defined functions.
This project converts standard engineering equations into clean, reusable Python functions for academic learning and practical understanding.

📘 Project Overview

This repository contains separate Python functions for calculating key parameters of the Ideal Otto Cycle, including:
Compression Ratio
Otto Cycle Efficiency
Net Work Output
Mean Effective Pressure (MEP)
Temperature Relations
Heat Addition and Rejection
Isentropic Relations
Each formula is implemented as an independent user-defined function for clarity and modular usage.


🛠 Technologies Used

Python 3
Math Module

🎯 Purpose of This Project

This project is designed for:
Mechanical Engineering students
Thermodynamics learners
Academic assignments and simulations
Strengthening programming skills in engineering applications

🚀 How to Use
Clone the repository.
Import the required function into your Python script.
Provide known input values.
Get the calculated thermodynamic parameter as output.
